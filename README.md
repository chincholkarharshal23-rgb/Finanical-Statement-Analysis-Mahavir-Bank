# Financial Statement Analysis - Shri Mahavir Urban Co-operative Bank, Solapur

## Objective
To analyze the financial performance of Shri Mahavir Urban Co-operative 
Bank, Solapur over a 5-year period using ratio analysis.

## Tools Used
- Microsoft Excel
- Ratio Analysis
- MS word

## Analysis Performed
- Balance Sheet Analysis
- Profit & Loss Statement Analysis
- Cash Flow Statement Analysis
- Liquidity Ratios
- Profitability Ratios
- Solvency Ratios

## Key Findings
- The bank showed consistent growth in deposits over 5 years
- Profitability ratios showed improvement from 2020 to 2024
- Identified areas of strength and improvement in financial performance

## What I Learned
- Real-world financial statement analysis
- Co-operative banking functions and structure
- How to identify financial strengths and weaknesses of a bank
